Road A Eye는 고속도로 CCTV 영상을 AI로 분석해 진입 불가 물체를 즉시 감지하고, 관리자에게 실시간 알람을 전송하는 스마트 관제 시스템입니다.

Overview

프로그램 개요

고속도로에 진입해서는 안 되는 물체를 CCTV 영상에서 자동으로 탐지하고 즉시 관제 인원에게 알립니다.

📷
📷
📷
📷
LIVE — 이상 물체 감지됨
🚨
실시간 이상물체 감지
킥보드, 오토바이, 보행자 등 고속도로 진입 금지 물체를 AI 모델이 자동으로 분류·감지합니다.
📡
관리자 실시간 알람
감지 즉시 관제 시스템 대시보드에 알림이 표시되며, 이메일 등 다채널 알람을 지원합니다.
🎥
CCTV 통합 관제
ITS(교통정보시스템) API와 연동해 고속도로 전국 CCTV를 통합 관리하고 MJPEG 스트리밍을 제공합니다.
🤖
AI 챗봇 지원
Claude AI 기반 챗봇으로 감지 현황·통계 조회 및 운영 문의를 자연어로 처리합니다.

Features

주요 기능

감지부터 관제, 통계, 운영까지 고속도로 안전 관리에 필요한 모든 기능을 제공합니다.

🚨

이상물체 실시간 탐지

AI 모델이 CCTV 영상을 프레임 단위로 분석해 킥보드·오토바이·보행자 등 고속도로 진입 금지 물체를 자동 분류합니다.

Core Feature
📡

실시간 관리자 알람

감지 이벤트 발생 즉시 관제 대시보드에 알람이 표시됩니다. 감지 이미지와 위치 정보가 함께 기록됩니다.

Real-time
🎥

CCTV 스트리밍 관제

RTSP/HTTP CCTV 스트림을 MJPEG으로 변환해 웹 브라우저에서 별도 플러그인 없이 실시간 영상을 확인할 수 있습니다.

MJPEG Stream
📊

감지 통계 및 분석

CCTV별·시간대별·물체 유형별 감지 현황을 통계로 확인합니다. 이상 패턴 조기 파악에 활용됩니다.

Analytics
🤖

AI 챗봇 (Claude)

Anthropic Claude API 기반 챗봇으로 감지 현황 조회, 운영 문의 등을 자연어로 처리합니다. 세션별 대화 히스토리를 유지합니다.

Claude AI
🗺

ITS 연동 및 CCTV 동기화

한국 교통정보시스템(ITS) API와 연동해 고속도로 CCTV 목록을 자동으로 동기화·관리합니다.

ITS API

이메일 인증 회원가입

이메일 인증 코드 발송 → 코드 확인 → 가입 완료의 3단계 인증으로 관제 인원 계정을 안전하게 관리합니다.

Security

공지사항, 문의, FAQ, 자료실 게시판을 통해 운영 정보를 체계적으로 관리하고 공유합니다.

DB 이중화 구조 (Keepalived VRRP)

VIP(Floating IP)는 게이트웨이가 아닌 같은 네트워크 안에서 서버 간에 이동하는 HA용 IP입니다. Keepalived가 2초마다 MySQL 상태를 체크하여 장애 감지 시 VIP를 자동으로 104(DB Server)로 이전합니다.

Software Architecture

소프트웨어 아키텍처

레이어드 아키텍처로 관심사를 분리해 유지보수성과 확장성을 높였습니다.

레이어 구조

⚛
Client Layer — React (192.168.0.103)
관제 대시보드 UI · CCTV 스트리밍 화면 · 알람 수신
React img src MJPEG
↕ HTTP / REST API
🔀
Router Layer — routers/
요청 파싱 · 응답 직렬화 · 인증(JWT Depends) · 비즈니스 로직은 Service에 위임
/auth /cctv /chat /board /models
↕ Python function call
⚙️
Service Layer — services/
비즈니스 로직 · DB 쿼리 · 외부 API 호출 (ITS, Claude, Gmail SMTP)
auth_service cctv_service stream_service chat_service
↕ async SQLAlchemy ORM
🗂
ORM Layer — models/
SQLAlchemy 테이블 정의 · Pydantic 스키마(schemas/) 로 요청/응답 검증
orm.py chat_orm.py model_orm.py board_orm.py
↕ aiomysql (비동기)
🗄
Database — MySQL (192.168.0.104)
road_a_eye DB · 서버 시작 시 create_all 자동 생성 · 금지 클래스 자동 시딩
